# RFID 기반 보안 시스템 설계 및 구현
</br>

### 프로젝트 설명📝
* NFC 태그 카드를 사용하여 등록된 사용자와 미등록된 사용자를 분류하는 프로젝트
* 등록된 사용자가 카드를 태그 할 경우 LCD화면에 사용자의 정보를 출력하고, 서브모터를 활용하여 잠금이 해제되는 기능을 구현
* 승인된 기기 이외의 연결 시도 시 차단 및 경고음 발생 기능 구현


</br>
